## Unzipped Parts
As the name suggests, unzipped `.fzpz` files.

## Naming Scheme
```
Manufacturer/
├── manufacturer_Part-Name/
|   ├── part.[moduleID].fzp
|   ├── svg.bb_[moduleID].svg
|   ├── svg.sch_[moduleID].svg
|   ├── svg.pcb_[moduleID].svg
|   └── svg.icon_[moduleID].svg <-- Optional in some parts
```
